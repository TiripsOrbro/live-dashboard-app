# Daily Food Safety Check (DFSC)

Audit definition: questions, schemas, and report builders for the daily food safety check.

- `dfscSchema.js` — question definitions and validation rules
- `dfscStore.js` — session persistence under `tacaudit/data/dfsc/`
- `dfscReport.js` — PDF report generation
- `dfscCoreReport.js` — combined CORE summary PDF
- `dfscAdmin.js` — admin compliance helpers
