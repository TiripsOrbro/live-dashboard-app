# Square One

Audit definition: Square One area questions and due-area scheduling.

- `squareOneSchema.js` — question definitions
- `squareOneAreas.js` — area rotation and due slots
- `squareOneQuestions.generated.json` — generated question bank
- `squareOneStore.js` — session data under `tacaudit/data/square-one/`
- `squareOneReport.js` — PDF report generation
